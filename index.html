GitHub

Download and install Git CLI

Test to see if Git CLI was successfully installed with git --version

Create a GitHub Account

Create a new repository

Set the name to 'my-first-plugin'

git init

IntelliJ

Download and install IntelliJ

Click on 'Create New Project'

Select 'Gradle' tab on left and select 'Java' under additional libraries and frameworks

Set the plugin name, the name should follow the CamelCaseFormat, e.g. 'MyFirstPlugin'

Set the groupId to your GitHub, e.g. 'com.github.valkyrienyanko'
